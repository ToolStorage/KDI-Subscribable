# Changelog

## [1.0.0] - 2026-05-14

### Added
- SubscribableProperty<T> reactive value with change notification
- SubscribableCollection<T> reactive list with Add/Remove/Replace/Move/Clear events
- SubscribableDictionary<TKey, TValue> reactive dictionary
- SubscribableCommand with conditional execution
- ReadOnlySubscribableProperty<T> wrapper
- LINQ extensions (Select, Where)
- CompositeDisposable and AddTo() extension
- Reaction transaction system for batched notifications
- Closure Profiler editor tool
- SubscribableProperty Debugger editor window
